---
description: Generate a tailored cover letter for a specific Job Description (JD).
---

1. **Prerequisite Check**:
    - Ensure you have the Job Description (JD) text.
    - Ensure `.github/resume-master-content.md` is accessible for fact-checking.

2. **Read Instructions**:
    - Read `.github/instructions/cover-letter-instructions.md` completely.
    - Read `.github/resume-master-content.md` completely.

3. **Analyze Job Description**:
    - Extract top 5 responsibilities and required skills.
    - Identify company vibes and mission for the "Why this company" paragraph.

4. **Planning**:
    - Identify 2-3 key strengths/achievements from master content that map to JD.
    - Draft the 4-paragraph structure:
        1. **Opening**: Hook, no "excited to apply".
        2. **What You Bring**: Evidence + Metrics.
        3. **Why This Company**: Specific connection.
        4. **Closing**: Confident sign-off.

5. **Generation**:
    - Generate the cover letter in Markdown.
    - **CRITICAL RULES**:
        - **One Page Max**: 250-300 words.
        - **No Hyphen Sentence Breaks**: Use commas or periods.
        - **No EM Dashes**.
        - **Tone**: Direct, confident, human. No clichés ("perfect fit", "honor").

6. **Self-Audit**:
    - Check against "Forbidden Phrases" in instructions.
    - Verify all metrics against master content.

7. **Output**:
    - Present the final Markdown code block.
